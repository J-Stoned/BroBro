What is Tabular View for Sub-Accounts List?
Tabular View transforms the Sub-Accounts page from a Card layout into a spreadsheet-style table. Each sub-account is displayed as a row, while columns surface details like Name, Status, Creation Date, Plan, and more. Agency users can flip between Card and Table layouts at any time, giving them the flexibility to scan high-level info or dive into sortable data when precision matters. Tabular view gives Agency Owners the ability to using Sorting and filtering to quickly locate specific Sub-Accounts, saving valuable time.

Centralized Visibility:
See every sub-account in a single, scrollable grid.

Switching Between Card View and Tabular View
HighLevel offers two interchangeable layouts on the Sub-Accounts page:
Card View: Best for visual snapshots (logos, quick stats).
Tabular View: Best for data comparison, bulk edits, and sorting.
To change between views, follow these steps:
Step 1:
Navigate to Sub-Accounts
From the Agency-View dashboard,  navigate to the Sub-Accounts page.
Step 2:
Change Layout
Use the layout button (Switch to Card/Table View) in the upper-right corner of the Sub-Accounts page to switch instantly. Your last-used view is remembered per user, so you always return to your preferred workflow.
Sorting and Filtering Sub-Accounts in Tabular View
Sorting provides one-click column ordering, ideal for alphabetizing by Name or ranking by Created Date. To sort accounts, click the
Sort
button and choose a category to sort based on from the drop down.
Using the
A-Z
/
Z-A
dropdown, you are able to choose between alphabetical and reverse alphabetical order.
Customizing Visible Columns
Within Tableview, you are able to select which columns are and are not shown, customizing the view to match your needs!
To select the visible columns, click the
Columns
button. Then from the drop down, use the checkboxes to select the columns you would like to be shown.
Frequently Asked Questions
Q: Does switching to Tabular View affect my data?
No. The view only changes how information is displayed. Your sub-account data remains untouched.
Q: Are filters remembered between sessions?
Filters reset when you leave the page; however Smart Lists remain at the top bar for easy access.
Q: Can I customize which columns appear?
Yes. Click the Columns icon to toggle visibility for each data field.
Q: Can I export reports from table view?
Yes! To export a report from table view, just click the Export button in the upper-right corner.
